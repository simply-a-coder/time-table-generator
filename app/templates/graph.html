Graph Visualization

This graph represents conflicts between courses:

  • Nodes (circles): Courses
  • Gray edges: Student conflicts (courses with shared students)
  • Red edges: Instructor conflicts (courses taught by the same instructor)
Graph Theory in Scheduling

The graph coloring algorithm uses the following approach:

  1. Each course is represented as a node in the graph
  2. Conflicts between courses are represented as edges
  3. The scheduler assigns "colors" (time slots) to each node
  4. No two adjacent nodes (courses with conflicts) can have the same color
  5. The goal is to use the minimum number of colors (time slots) while satisfying all constraints

The scheduler prioritizes courses with the most conflicts (highest degree nodes) to improve efficiency.
